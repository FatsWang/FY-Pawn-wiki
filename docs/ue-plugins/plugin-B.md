## 🔧 安装与配置

### 系统要求
- 虚幻引擎版本：5.0 及以上（或指定版本）
- 操作系统：Windows / Mac / Linux

### 安装步骤

1. 下载插件文件
2. 将 `Plugin-B` 文件夹复制到项目的 `Plugins` 目录
3. 关闭虚幻编辑器
4. 右键点击 `.uproject` 文件，选择"Generate Visual Studio project files"
5. 重新打开虚幻编辑器，编译插件
6. 在 Edit > Plugins 中搜索插件名称并启用
